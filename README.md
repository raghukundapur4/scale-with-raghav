# Scale With Raghav

> **Build. Automate. Scale.**

A modern, premium, responsive landing page for Scale With Raghav — helping coaches and businesses grow using high-converting websites, WhatsApp automation, and lead generation systems.

## 🚀 Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- GitHub Pages

## 📦 Getting Started

```bash
npm install
npm run dev
```

## 🏗️ Build & Deploy

```bash
npm run build
npm run deploy
```

The site will be live at: https://raghukundapur4.github.io/scale-with-raghav/
